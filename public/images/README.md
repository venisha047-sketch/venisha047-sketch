# Image system

Drop the three world anchor images into this folder. The CSS references them by these exact filenames:

- `generated-image-10.jpg` — The Door scene (portal entry)
- `generated-image-9.jpg` — World map blueprint (used in the hub)
- `generated-image-7.jpg` — World reveal atmosphere (floating-city horizon)

If files are missing, the scenes still render — each has a layered gradient + particle fallback so the experience never collapses to a blank screen.
